# backend/db/data/
#
# This directory holds data files for the AISHE + NIRF seeding pipeline.
#
# REQUIRED FILES (you must download these manually):
#
#   aishe_colleges.xlsx   ← AISHE institutional directory
#     Download from: https://aishe.gov.in/ → Reports → AISHE Final Report
#     or from:       https://aikosh.indiaai.gov.in/ (search "AISHE Institutional Directory")
#
#   nirf_rankings.csv     ← NIRF 2024 rankings (all categories)
#     Download from Kaggle: search "NIRF Rankings 2024 combined"
#     or compile manually from: https://www.nirfindia.org/2024/
#
# OUTPUT FILES (generated by the seed scripts — safe to delete and regenerate):
#
#   review_ambiguous.txt      ← colleges assigned Science stream by default (for manual review)
#   filtered_out_sample.txt   ← 30-row random sample of colleges excluded by filtering rules
#   nirf_fuzzy_matches.txt    ← NIRF matches with confidence < 85% (for manual review)
#   nirf_unmatched.txt        ← NIRF colleges that couldn't be matched to any AISHE row
#
# DATA FILES ARE GITIGNORED (large binary/CSV files should not be committed).
